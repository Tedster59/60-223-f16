Physical Computing Resources

This repository 'physcomp-f15' provides course materials for several courses in Physical Computing as part of the IDeATe program at Carnegie Mellon University. For details, see the IDeATe web site and the main program site.

Contents

The Laboratory Exercise area contains many self-contained demonstrations and experiments, including sample code and circuits.
