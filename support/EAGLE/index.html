Physical Computing EAGLE Libraries

EAGLE is an electronic CAD package which supports schematic capture and circuit board design. The 'Light' edition is available for free and is usable for small boards (up to 100x80 mm) and single-sheet schematics.

EAGLE is available from cadSoft:

  • http://www.cadsoftusa.com/
  • http://www.cadsoftusa.com/download-eagle/freeware/

EAGLE includes libraries which define many common electronic parts, but there are also many third-party libraries. Some useful sources for EAGLE libraries are as follows:

  • https://github.com/sparkfun/SparkFun-Eagle-Libraries
  • https://github.com/adafruit/Adafruit-Eagle-Library
  • http://www.element14.com/community/community/cadsoft_eagle/eagle_cad_libraries
