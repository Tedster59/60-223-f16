Physical Computing Pure Data References

Pure Data (Pd) is a dataflow programming language which supports real-time event and audio processing. It is free and open-source.

Contents

  • reference-help.pd : Pure Data patch to serve as an index for the following.
  • vocabulary.pd : Pure Data patch listing the core language operators.
  • advanced-vocabulary.pd : Pure Data patch listing more esoteric standard operators.
  • externals-vocabulary.pd : Pure Data patch listing all 'external' (non-core) operators used in the course exercises.
